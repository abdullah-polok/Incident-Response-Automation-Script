# Incident Response and Threat Automation Tool

A python-based command-line tool that automates system artifact collection, process scanning,anomaly detection,file integrity checks,and forensic reporting.

# Show the banner and help menu

python main.py

# Run a full scan

python main.py --full

# Collect only system information

python main.py --system

# Scan network connections

python main.py --network

# Hash a specific file

python main.py --hash /path/to/file
